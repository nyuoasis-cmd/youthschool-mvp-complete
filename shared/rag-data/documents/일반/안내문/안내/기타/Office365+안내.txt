오피스 제품(파워포인트, 엑셀, 워드) 다운로드 설치 제공 사용대상 : 인천 관내 교직원, 학생(업무용SW 보급기관에 한함) 사용자 ID 당 최대 5대 디바이스 설치 가능 클라우드 오피스 제공 : 인터넷상에서 문서 작업 후 저장 활용 가능 클라우드 스토리(OneDrive) 제공 : 사용자당 1TB 메일함 제공 : 사용자당 50GB 제공(수/발신함) 기타 서비스 제공 : 공유 캘린더, 주소록 등 가입 방법 퀀됀X 크롬 혹은 엣지(Edge)로 접속) 약관 동의 및 사용자 인증 절차 알림톡을 통한 사용자 인증 인증 코드 : 홈페이지-학부모마당 게시판에 탑재함 기본 정보 입력 : 소속학교(기관), 이름, 입학년도(학생) 계정정보 입력 : 아이디, 암호 아이디@o365.ice.go.kr로 계정생성) 인증 코드 외부 유출시 불법 사용이 되므로, 유출되지 않도록 유의 뛈돩돩 뉾???? 제품 사용 문의 온라인 : 가입시스템 오른쪽 상단 Q&A 이용 오프라인: 에이마스 02-6123-4124
