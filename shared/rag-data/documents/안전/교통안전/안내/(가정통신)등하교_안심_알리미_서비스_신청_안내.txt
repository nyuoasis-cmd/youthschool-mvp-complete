담당:체육안전부(이상준) 02-490-2500 제2025-026호 등하교 안심 알리미 서비스 신청 안내 (신청한 등하교 단말기는 집으로 택배 발송됩니다) (신청 방법 및 APP 설치 안내 / 뒷면 필독) 학부모님 안녕하십니까? 본교에서는 우리 자녀들의 안전한 등하교길 조성을 위하여 “등하교 안심 알리미 서비 스 (T-안심 알리미)”를 시행하고 있습니다. 학생들이 가방에 단말기를 휴대하고 다니 면, 학교 교문마다 중계기 센서가 설치되어 있어, 자녀의 등하교 상황을 메시지(APP) 로 수신 가능합니다. 서비스는 희망자 선택이며, 신청 시 단말기는 댁으로 택배 발송 됩니다. 단말기 수령 시 동봉된 사용설명서와 같이 단말기 번호를 직접 등록 후 가방 에 착용해 주시기 바랍니다. 학교 교문 통과 시 자동으로 등하교 알림 서비스가 시작 됩니다. * 신청기간 : 2025년 3월11일(화) ~ 3월 20일(목)까지 1) T-안심알리미 서비스 내용 (가방 중앙 외부에 걸어 주세요) 1. 등하교 시간에 실시간으로 문자(앱) 발송을 합니다. - 등교 : 08:30 홍길동 학생이 등교하였습니다. (정문/후문)**초등학교 - 하교 : 14:00 홍길동 학생이 하교하였습니다. (정문/후문)**초등학교 - 하교 후 : 15:00홍길동 학생의 신호가 인식되었습니다. **초등학교 2. 등하교 안심 알리미 서비스는 교문(현관 등) 통과 시 자동으로 발송됩니다. - 교문(중계기) 근처에 가방을 두고, 학생은 다른 곳에 있을 경우 감지되어 부모님께 알림 메시지가 발송되오니 이점 꼭 유의하시기 바랍니다. (주의) 3. 방학 기간에도 서비스는 정상 제공됩니다. 4. 알림 수신은 주보호자 포함 최대 6명까지 가능합니다. (엄마,아빠,할머니등) 2025. 3. 11. 서 울 새 솔 초 등 학 교 장 <부패신고 신고제도 안내문> ◈ 관계 교직원이 금품‧향응‧편의제공 등을 요구할 경우에는 아래로 신고하여 주시기 바랍니다. ☞ 홈페이지: 서울시교육청 공익제보센터 ☞ 스마트폰 활용 신고:왼쪽 QR코드 인식 > 공익제보센터 2) T-안심 알리미 신청 안내 (상시 가능 / 뒷면 참조 하세요) 1. 상품명 1년 상품(1년 정액 요금)후불 상품(월 요금) 2. 이용료 ① 이용료 : 33,000원 / 1년 ② 배송료 : 3,000원 (수익자부담) ③ 단말기 구매비 : 0원 (1회 무상) - 분실 시 직접 구매 필요 ① 이용료 : 3,300원 / 1개월 ② 배송료 : 3,000원 (수익자부담) ③ 단말기 구매비 : 10,000원 (1회성) - 분실 시 직접 구매 필요 3. 신청기간 -3월4일~ 상시 신청가능-3월22일~ 상시 신청가능 * 신청 시 이벤트(할인) 상품이 있다면, 선택할 수 있습니다.
