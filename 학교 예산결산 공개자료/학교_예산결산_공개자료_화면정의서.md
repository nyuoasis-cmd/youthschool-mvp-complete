# 학교 예산/결산 공개 자료 화면정의서

## 1. 화면 구조 개요

### 1.1 전체 화면 흐름
```
[홈페이지] 
    ↓
[문서 유형 선택 - 예산/결산 공개 자료 카드 클릭]
    ↓
[작성 페이지 - STEP 1~8 단계별 입력]
    ↓
[미리보기 페이지]
    ↓
[생성 완료 페이지]
```

## 2. 홈페이지 - 문서 카드 추가

### 2.1 화면 ID
`HOME-007`

### 2.2 화면 경로
`/` (홈페이지)

### 2.3 추가 요소
**예산/결산 공개 자료 카드**

```
┌─────────────────────────────────────────┐
│  💰 예산/결산 공개 자료                  │
│                                          │
│  학교 예산 및 결산 공개 설명 자료를      │
│  전문적으로 작성합니다.                  │

└─────────────────────────────────────────┘
```

### 2.4 컴포넌트 속성
| 속성 | 값 |
|------|-----|
| 아이콘 | 💰 (돈 주머니) |
| 제목 | 예산/결산 공개 자료 |
| 설명 | 학교 예산 및 결산 공개 설명 자료를 전문적으로 작성합니다. |
| 예시 태그 | "예산/결산 구분 지원", "재원별 자동 계산", "인포그래픽 생성" |
| 링크 | `/create/budget-disclosure` |

### 2.5 배치 위치
- 기존 "학교운영위원회 회의록" 카드 다음
- "외부 교육 용역 계획서" 카드 이전
- 4번째 카드 위치

## 20. 참고사항

### 20.1 기존 문서 유형과의 일관성
- 전체 레이아웃 구조 동일
- 단계별 입력 방식 동일
- AI 생성 프로세스 동일
- 버튼 스타일 및 배치 동일

### 20.2 특수 기능
- 예산/결산 구분에 따른 조건부 표시
- 재원별/사업별 자동 계산 (금액, 비율, 총계)
- 학생 1인당 금액 자동 계산
- 집행률 자동 계산 (결산)
- 4개 동적 테이블 (재원, 사업, 분기별, 주요사업 상세)
- 예산: 11개 기본 + 사업별 3개씩 AI 버튼
- 결산: 15개 기본 + 사업별 3개씩 AI 버튼
- 4개 붙임 문서 자동 생성 (표 2개, 인포그래픽 2개)

### 20.3 확장 가능성
- 전년도 예산/결산 비교 기능
- 타 학교 평균 대비 분석
- 월별 집행 현황 추적
- 예산 시뮬레이션 기능
- 차트/그래프 커스터마이징
