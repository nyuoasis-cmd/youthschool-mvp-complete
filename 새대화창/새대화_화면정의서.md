## 1. 화면 구조 개요

### 1.1 전체 레이아웃

```
┌─────────────────────────────────────────────────────────────────────┐
│                          [상단 네비게이션 바]                          │
├────────────┬────────────────────────────────────────────────────────┤
│            │                    [메인 컨텐츠 영역]                    │
│            │                                                          │
│  [좌측     │                    [대화 헤더]                           │
│   사이드바] │  ┌────────────────────────────────────────────────┐   │
│            │  │                                                  │   │
│  - 새 대화  │  │                                                  │   │
│  - 검색     │  │              [대화 내용 영역]                    │   │
│  - 대화목록 │  │                                                  │   │
│            │  │                                                  │   │
│            │  └────────────────────────────────────────────────┘   │
│            │                                                          │
│            │                  [입력 영역]                             │
└────────────┴────────────────────────────────────────────────────────┘

[좌측: 260px] [우측: Flexible]
[총 높이: 100vh]
```

### 1.2 화면 영역별 역할

| 영역 | 크기 | 역할 |
|------|------|------|
| 상단 네비게이션 | 전체 × 64px | 로고, 사용자 정보, 설정 |
| 좌측 사이드바 | 260px × (100vh-64px) | 대화 목록, 검색, 새 대화 |
| 메인 컨텐츠 | Flexible × (100vh-64px) | 대화 화면 |
| 대화 헤더 | 100% × 60px | 대화 정보, 메뉴 |
| 대화 내용 | 100% × Flexible | 메시지 표시 |
| 입력 영역 | 100% × Auto (최소 80px) | 메시지 입력, 첨부 |

## 2. 상세 화면 정의

### 2.1 상단 네비게이션 바

#### 2.1.1 레이아웃

```
┌─────────────────────────────────────────────────────────────────────┐
│ ☰  [로고] YouthSchool          [🔔] [⚙️] [👤 김대리 ▼]             │
└─────────────────────────────────────────────────────────────────────┘
높이: 64px
배경: #FFFFFF
하단 테두리: 1px solid #E0E0E0
```

#### 2.1.2 구성 요소

**좌측 영역**
```
☰  [로고] YouthSchool
[24px] [32px] [텍스트]
```
- **햅버거 메뉴 (☰)**: 모바일에서만 표시
  - 크기: 24px × 24px
  - 색상: #424242
  - 클릭: 사이드바 토글
  
- **로고**: 
  - 크기: 32px × 32px
  - 이미지: youthschool-logo.svg
  - 클릭: 홈으로 이동

- **서비스명**:
  - 폰트: Pretendard Bold 18px
  - 색상: #1976D2
  - 간격: 로고에서 12px

**우측 영역**
```
[🔔] [⚙️] [👤 김대리 ▼]
[32px] [32px] [프로필]
```
- **알림 버튼 (🔔)**:
  - 크기: 32px × 32px
  - 아이콘: bell-outline
  - 뱃지: 읽지 않은 알림 수
  - 클릭: 알림 드롭다운

- **설정 버튼 (⚙️)**:
  - 크기: 32px × 32px
  - 아이콘: settings-outline
  - 클릭: 설정 페이지로 이동

- **프로필 드롭다운**:
  - 아바타: 32px × 32px 원형
  - 이름: Pretendard Medium 14px
  - 드롭다운 화살표: 8px
  - 클릭: 메뉴 펼침

**프로필 드롭다운 메뉴**
```
┌───────────────────────┐
│ 👤 김대리             │
│ nyuoasis@gmail.com    │
```
