## 2. 공통 컴포넌트

### 2.1 헤더 (Header)

#### 2.1.1 데스크톱 헤더 (≥ 768px)
```
┌───────────────────────────────────────────────────┐
│                                                   │
│  [⭐ 티처메이트]                     [로그인 →]   │
│                                                   │
└───────────────────────────────────────────────────┘
```

**스펙**:
- 높이: 56px
- 배경색: #FFFFFF
- 하단 테두리: 1px solid #E5E7EB
- position: sticky, top: 0
- z-index: 1000

**요소**:
1. **로고 + 서비스명**
   - 위치: 좌측, padding-left: 24px
   - 로고 크기: 24x24px
   - 텍스트: "티처메이트"
   - 폰트: 18px, font-weight: 600
   - 색상: #1F2937
   - 클릭 시: 홈으로 이동

2. **로그인 버튼**
   - 위치: 우측, padding-right: 24px
   - 크기: auto x 36px
   - padding: 8px 16px
   - 배경색: #3B82F6
   - 텍스트: "로그인"
   - 색상: #FFFFFF
   - border-radius: 6px
   - 호버: 배경색 #2563EB
   - 활성: 배경색 #1D4ED8

#### 2.1.2 모바일 헤더 (< 768px)
```
┌──────────────────────────────┐
│                              │
│  [☰]  [⭐ 티처메이트] [로그인] │
│                              │
└──────────────────────────────┘
```

**스펙**:
- 높이: 56px
- 레이아웃: flex, justify-content: space-between

**요소**:
1. **햄버거 버튼**
   - 위치: 좌측, padding-left: 12px
   - 아이콘 크기: 24x24px
   - 터치 영역: 48x48px
   - 색상: #4B5563
   - aria-label: "메뉴 열기"

2. **로고 + 서비스명**
   - 위치: 중앙
   - 로고 크기: 20x20px
   - 텍스트: 16px, font-weight: 600

3. **로그인 버튼**
   - 위치: 우측, padding-right: 12px
   - 크기: auto x 32px
   - padding: 6px 12px
   - 폰트 크기: 14px

### 2.3 입력 영역 (Input Area)

#### 2.3.1 데스크톱
```
┌─────────────────────────────────────────────────┐
│  [📎]  [무엇을 도와드릴까요?____________]  [↑]  │
└─────────────────────────────────────────────────┘
```

**스펙**:
- 위치: 하단 고정 OR 메인 영역 하단
- 전체 너비: 100%
- 최소 높이: 56px
- 패딩: 12px 16px
- 배경색: #FFFFFF
- 테두리: 1px solid #E5E7EB
- border-radius: 12px

**요소**:
1. **파일 첨부 버튼**
   - 크기: 40x40px
   - 아이콘: 📎 (24x24px)
   - 색상: #6B7280
   - 호버: 색상 #4B5563

2. **텍스트 입력창**
   - 너비: flex-grow: 1
   - 최소 높이: 40px
   - 최대 높이: 120px
   - padding: 8px 12px
   - 폰트 크기: 16px
   - placeholder: "무엇을 도와드릴까요?"
   - resize: vertical (자동 높이 조정)

3. **전송 버튼**
   - 크기: 40x40px
   - 아이콘: ↑ (24x24px)
   - 배경색: #3B82F6
   - 색상: #FFFFFF
   - border-radius: 8px
   - 비활성 (입력 없을 때): 투명도 50%
   - 활성: 투명도 100%, 호버 #2563EB

#### 2.3.2 모바일
```
┌──────────────────────────────┐
│  [📎]  [입력창_______]  [↑]  │
└──────────────────────────────┘
```

**스펙**:
- 위치: 하단 고정 (position: fixed, bottom: 0)
- bottom: safe-area-inset-bottom (iOS 노치 대응)
- 패딩: 8px 12px

**요소**:
- 파일 첨부: 44x44px
- 입력창: flex-grow: 1, 최소 44px 높이
- 전송: 44x44px
