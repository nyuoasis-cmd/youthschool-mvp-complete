# 학부모총회 안내 화면정의서

## 1. 화면 구조 개요

### 1.1 전체 화면 흐름
```
[홈페이지] 
    ↓
[문서 유형 선택 - 학부모총회 안내 카드 클릭]
    ↓
[작성 페이지 - STEP 1~6 단계별 입력]
    ↓
[미리보기 페이지]
    ↓
[생성 완료 페이지]
```

## 2. 홈페이지 - 문서 카드 추가

### 2.1 화면 ID
`HOME-005`

### 2.2 화면 경로
`/` (홈페이지)

### 2.3 추가 요소
**학부모총회 안내 카드**

```
┌─────────────────────────────────────────┐
│  👥 학부모총회 안내                      │
│                                          │
│  학기 초 학부모총회 안내문을             │
│  전문적으로 작성합니다.                  │
│                                          │
│  ✓ 정기/임시총회 지원                   │
│  ✓ 안건별 상세 작성                     │
│  ✓ 참석 안내 자동 생성                  │
│                                          │
│  [작성하기 →]                            │
└─────────────────────────────────────────┘
```

### 2.4 컴포넌트 속성
| 속성 | 값 |
|------|-----|
| 아이콘 | 👥 (사람들) |
| 제목 | 학부모총회 안내 |
| 설명 | 학기 초 학부모총회 안내문을 전문적으로 작성합니다. |
| 예시 태그 | "정기/임시총회 지원", "안건별 상세 작성", "참석 안내 자동 생성" |
| 링크 | `/create/parent-meeting` |

### 2.5 배치 위치
- 기존 "가정통신문" 카드 다음
- "외부 교육 용역 계획서" 카드 이전
- 2번째 카드 위치
