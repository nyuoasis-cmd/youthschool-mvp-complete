# 학교폭력 예방 교육 계획서 화면정의서

## 1. 화면 구조 개요

### 1.1 전체 화면 흐름
```
[홈페이지] 
    ↓
[문서 유형 선택 - 학교폭력 예방 교육 계획서 카드 클릭]
    ↓
[작성 페이지 - STEP 1~10 단계별 입력]
    ↓
[미리보기 페이지]
    ↓
[생성 완료 페이지]
```

### 3.4 공통 UI 요소

#### 헤더
- 좌측: 뒤로가기 버튼 (← 아이콘)
- 중앙: 페이지 제목 "학교폭력 예방 교육 계획서 작성"
- 우측: 임시저장 버튼

#### 좌측 사이드바 (Stepper)
- 10개 단계 표시
- 현재 단계 하이라이트 (파란색)
- 완료 단계 체크 표시 (✓)
- 미완료 단계 회색 표시

#### 하단 버튼
- 좌측: "이전" 버튼 (첫 단계에서는 비활성화)
- 우측: "다음 단계" 버튼 (마지막 단계에서는 "생성하기")

## 14. AI 답변 생성 버튼 공통 기능

### 14.1 버튼 UI 및 동작

#### 버튼 상태
```
기본 상태:    [AI 답변 생성]  (파란색 버튼, #2563eb)
로딩 중:      [생성 중... ⏳]  (회색, 비활성화, #94a3b8)
완료:         [AI 답변 생성]  (원상 복구)
```

#### 동작 흐름
1. 사용자가 "AI 답변 생성" 버튼 클릭
2. 버튼 상태 변경: "생성 중... ⏳" (비활성화)
3. 현재까지 입력된 모든 컨텍스트를 API로 전송
4. AI 생성 응답 수신 (5-10초)
5. 텍스트영역에 생성된 내용 자동 삽입
6. 버튼 원상 복구
7. 사용자가 생성된 내용을 검토하고 수정 가능

#### 에러 처리
| 에러 상황 | 알림 메시지 | 버튼 상태 |
|----------|------------|----------|
| API 호출 실패 | "생성 실패. 다시 시도해 주세요." | 원상 복구 |
| 네트워크 오류 | "네트워크 연결을 확인해 주세요." | 원상 복구 |
| 타임아웃 (30초) | "시간이 초과되었습니다. 다시 시도해 주세요." | 원상 복구 |
