## 참고 문서 URL 수집 사용법

이 기능은 공개된 문서 URL을 수집해서 AI 생성에 참고할 수 있도록 저장합니다.
수집된 문서는 "참고 문서"로 등록되어 RAG 컨텍스트에 사용됩니다.

### 사용 방법 (비개발자용)
1) 운영자 계정으로 로그인합니다.
2) 관리자 화면(`/admin`)의 "문서 수집" 탭으로 이동합니다.
3) URL을 입력하고 "URL 수집" 버튼을 누릅니다.
4) "수집 완료" 메시지가 나오면 완료입니다.
5) 이후 일반 사용자 문서 생성 시 참고 문서가 자동 반영됩니다.

### 운영자 계정 생성
1) 관리자 화면(`/admin`)의 "운영자 관리" 탭으로 이동합니다.
2) 이메일/이름/휴대폰/비밀번호를 입력합니다.
3) "운영자 생성" 버튼을 누릅니다.

### 참고 사항
- 운영자 계정만 수집할 수 있습니다.
- 공개 페이지(URL)만 수집 가능합니다.
- 첨부파일은 목록으로 저장되며, HWP는 가능하면 텍스트를 추출합니다.
- PDF/DOCX 텍스트 추출은 추가 개발이 필요합니다.
