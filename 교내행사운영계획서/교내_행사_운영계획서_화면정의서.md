# 교내 행사 운영계획서 화면정의서

## 1. 화면 구조 개요

### 1.1 전체 화면 흐름
```
[홈페이지] 
    ↓
[문서 유형 선택 - 교내 행사 운영계획서 카드 클릭]
    ↓
[작성 페이지 - STEP 1~8 단계별 입력]
    ↓
[미리보기 페이지]
    ↓
[생성 완료 페이지]
```

---

## 2. 홈페이지 - 문서 카드 추가

### 2.1 화면 ID
`HOME-004`

### 2.2 화면 경로
`/` (홈페이지)

### 2.3 추가 요소
**교내 행사 운영계획서 카드**

```
┌─────────────────────────────────────────┐
│  📅 교내 행사 운영계획서                 │
│                                          │
│  체육대회, 축제, 졸업식 등 다양한        │
│  교내 행사 계획서를 자동 생성합니다.     │
│                                          │
│  ✓ 6가지 행사 유형 지원                 │
│  ✓ 안전 관리 체계 수립                  │
│  ✓ 예산 자동 계산                       │
│                                          │
│  [작성하기 →]                            │
└─────────────────────────────────────────┘
```

### 2.4 컴포넌트 속성
| 속성 | 값 |
|------|-----|
| 아이콘 | 📅 (달력) |
| 제목 | 교내 행사 운영계획서 |
| 설명 | 체육대회, 축제, 졸업식 등 다양한 교내 행사 계획서를 자동 생성합니다. |
| 예시 태그 | "6가지 행사 유형 지원", "안전 관리 체계 수립", "예산 자동 계산" |
| 링크 | `/create/event-plan` |

### 2.5 배치 위치
- 기존 "학교폭력 예방 교육 계획서" 카드 다음
- "HWP 양식으로 작성" 카드 이전
- 8번째 카드 위치
